# Golden Example Source

## Decision Capture

The user clarified that the new planner method should not be defined by
reading or projecting existing Novel Harness outlines. Those outlines are
beat-shaped artifacts from earlier architecture and may encode the wrong
assumptions.

## Rule

Golden examples for the planner output contract should come from authored
craft/template structures and corpus-derived distributions, such as:

- commercial chapter-function structures;
- romance obligatory beats;
- pulp quarter structure;
- LitRPG/progression strategy;
- Salvatore or other corpus-derived scene/function distributions once
  calibrated.

Existing harness outlines may be used only as baseline or migration evidence:
they can show how far legacy artifacts are from the target method, but they
must not define the target contract.
